# 3.5 接驳与入住交付指挥页 PRD

## Purpose
提供接驳车辆可视化、到店迎接队列管理与入住办理支持，保证交付环节顺畅可控。

## Target Users
- 调度、前台、礼宾、车队管理

## Scope
- GPS/司机上报、到店队列、入住办理、场景启动与异常改派

## Inputs
- 车辆位置、司机状态、预计到达时间、房态

## Outputs
- dispatch_task、service_delivery_record、入住所需动作

## Flow
1. 接驳任务生成 -> 2. GPS/司机上报 -> 3. 到店队列管理 -> 4. 入住办理与场景启动

## UI (ASC2)
- 左侧地图、右侧到店队列、入住办理表格、操作按钮

## Exceptions
- 偏航报警立即触发改派建议，房态冲突提示替代房

## Acceptance Criteria
- 车辆可视化与到店队列同步性良好，改派建议及时准确
