# 3.8 运营中台总览页 PRD

## Purpose
为运营/管理层提供关键KPI、预警看板、业务模块导航与下钻分析能力，支持快速定位运营问题并创建专项工单。

## Target Users
- 运营、管理、BI分析师

## Scope
- 指标卡、预警卡、业务模块入口、图表区与多维下钻

## Inputs
- 业务事件、核销/交付数据、渠道数据、财务数据

## Outputs
- 指标快照、专项工单、导出报告

## Flow
1. 数据聚合 -> 2. 指标计算 -> 3. 预警触发 -> 4. 工单创建/报表导出

## UI (ASC2)
- 指标卡、预警卡、业务模块导航、图表区

## Exceptions
- 指标计算异常需报警并提供重跑入口

## Acceptance Criteria
- 指标数据与源系统对齐，支持按渠道/场景/时间下钻
