# 3.9 权限与合规控制台 PRD

## Purpose
保障医疗/客户数据合规存取，集中管理角色权限、脱敏策略与审计检索，快速响应风险告警。

## Target Users
- 安全与合规、系统管理员、审计员

## Scope
- 权限矩阵、脱敏策略管理、审计日志检索、风险告警列表

## Inputs
- 角色定义、策略版本、审计事件流

## Outputs
- 权限变更记录、审计导出、告警事件

## Flow
1. 角色/策略配置 -> 2. 权限分配 -> 3. 审计事件记录 -> 4. 告警与升级

## UI (ASC2)
- 权限矩阵、脱敏策略列表、审计检索表、风险告警列表

## Exceptions
- 未授权访问立即阻断并触发审计通知

## Acceptance Criteria
- 审计查询性能满足SLA，权限变更有完整审计链
