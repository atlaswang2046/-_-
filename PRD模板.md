# 任务生成与配置 - 产品设计方案 V2.2

> 文档版本：V2.2  
> 创建日期：2026年1月6日  
> 更新说明：对齐现有系统设计规范（患者中心、任务中心、服务对照表）  
> 所属模块：患者中心 / 任务中心 / 系统配置

---

## 一、版本更新说明

### V2.2 更新内容

| 更新项 | 说明 |
|--------|------|
| 页面布局规范 | 统一采用左侧导航+顶部Tab+右侧抽屉的交互模式 |
| 状态流转优化 | 完善为：待纳入→已纳入→管理中→已结案 |
| 任务类型分类 | 对齐任务中心的分类体系（专项类/复诊类/专家类等） |
| 术语统一 | 物料→权益项目、任务类型→服务项、方案订单→健康方案 |
| 纳入配置完善 | 新增服务频次设置、收费确认、周期配置 |
| 服务对照表对齐 | 权益编码体系与ERP物料映射规范 |

### 4.2 权益项目计量类型

```
┌─────────────────────────────────────────────────────────────────────────────────────┐
│                              权益项目计量类型                                        │
├─────────────────────────────────────────────────────────────────────────────────────┤
│                                                                                     │
│   ┌─────────────────────────────────────────────────────────────────────────────┐   │
│   │                           1. 次数型 (COUNT)                                 │   │
│   ├─────────────────────────────────────────────────────────────────────────────┤   │
│   │  计量方式：按次数计量，每次完成扣减1次                                       │   │
│   │  典型权益：专家视频问诊×4次、月度回访×12次、季度评估×4次                     │   │
│   │  任务生成：纳入时生成首次任务，后续按周期自动生成                             │   │
│   │  核销方式：任务录入完成后自动扣减                                            │   │
│   └─────────────────────────────────────────────────────────────────────────────┘   │
│                                                                                     │
│   ┌─────────────────────────────────────────────────────────────────────────────┐   │
│   │                           2. 周期型 (PERIOD)                                │   │
│   ├─────────────────────────────────────────────────────────────────────────────┤   │
│   │  计量方式：按时间周期计量（天/月/年）                                        │   │
│   │  典型权益：托养康复管家×365天、专属健康管理×12个月                           │   │
│   │  任务生成：按周期自动生成定期任务（如每月生成1个回访任务）                    │   │
│   │  核销方式：时间到期自动完成，或按实际服务次数累计                             │   │
│   └─────────────────────────────────────────────────────────────────────────────┘   │
│                                                                                     │
│   ┌─────────────────────────────────────────────────────────────────────────────┐   │
│   │                           3. 重量型 (WEIGHT)                                │   │
│   ├─────────────────────────────────────────────────────────────────────────────┤   │
│   │  计量方式：按重量计量（KG），每次发放扣减对应重量                             │   │
│   │  典型权益：黄金量50KG（可兑换各种农产品）                                    │   │
│   │  任务生成：纳入时生成额度记录，管家按需创建发放任务                           │   │
│   │  核销方式：管家选择产品和数量，系统自动计算并扣减重量                         │   │
│   └─────────────────────────────────────────────────────────────────────────────┘   │
│                                                                                     │
│   ┌─────────────────────────────────────────────────────────────────────────────┐   │
│   │                           4. 实物型 (GOODS)                                 │   │
│   ├─────────────────────────────────────────────────────────────────────────────┤   │
│   │  计量方式：按件/台计量，一次性发放                                           │   │
│   │  典型权益：智能健康指环×1台、智能硬件(Huna)×1台                              │   │
│   │  任务生成：纳入时生成1个发货任务                                             │   │
│   │  核销方式：发货完成后自动核销                                                │   │
│   └─────────────────────────────────────────────────────────────────────────────┘   │
│                                                                                     │
└─────────────────────────────────────────────────────────────────────────────────────┘
```
