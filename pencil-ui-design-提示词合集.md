# pencil-ui-design 调用提示词合集

## 1. 全流程建页
调用 $pencil-ui-design，基于当前 .pen 文件，先初始化设计变量，再创建可复用组件库（Button/Card/Input/Badge），最后搭建一个B端列表页（筛选区+表格+分页），并截图自检。

## 2. 只做视觉统一
使用 pencil-ui-design 规范，统一当前文件所有页面的圆角、边框、阴影、字体层级和按钮样式，保持布局不变。

## 3. 新建组件并复用
调用 $pencil-ui-design，创建 Primary/Secondary 按钮和 Card 组件（reusable: true），并在 P2.1、P2.2 页面用 ref 批量替换旧按钮。

## 4. 表单页优化
按 pencil-ui-design 规范优化 P2.2 表单：双列布局、统一输入框样式、补充必填标识与说明文案，完成后截图检查。

## 5. 批量改样式
使用 pencil-ui-design 的批量更新方式，把所有卡片边框改为 #E4E4E7、圆角改为 12、按钮圆角改为 8，并输出变更清单。

## 6. 图标与图片
调用 $pencil-ui-design，为当前页面替换为 Material Symbols Rounded 图标，并用 G() 生成 3 张风格一致的 AI 插图。

## 推荐开场句
调用 $pencil-ui-design，按工业级规范改造我当前打开的页面。先做变量和组件标准化，再执行页面改造，最后截图验收。
